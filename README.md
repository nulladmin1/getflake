# `getflake`

Easily fetch my Nix Flake Templates using this `CLI` tool

To run without installing:

```nix
nix run github:nulladmin1/getflake
```

## Run (after cloning)

### Using `Nix` (recommended)

```bash
nix run
```

### Using `Cargo`

```bash
cargo run
```
